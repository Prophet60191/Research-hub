# The Research Hub
**Standardized Infrastructure for Agentic Research**
**Repository:** [https://github.com/Prophet60191/Research-Template](https://github.com/Prophet60191/Research-Template)

## Overview
This repository contains professional-grade templates for AI-assisted research. Each template provides a complete directory structure, role-specific instructions, and quality control protocols.

## Available Templates

### 1. Dataset Analysis (`Templates/Dataset_Analysis_Template`)
* **Purpose:** Quantitative research, data cleaning, and statistical reporting.
* **Engine:** Python (Pandas/Matplotlib).
* **Key Features:**
    * Strict "Read-Only" Raw Data protection.
    * "Document First, Code Second" workflow.
    * Reproducible "Findings" reports.
* **Best For:** Analyzing CSV/Excel files, academic research, data journalism.

### 2. Argument Map (`Templates/Argument_Map_Template`)
* **Purpose:** Logic auditing, evidence-based argumentation, and policy analysis.
* **Engine:** Admiralty System (NATO STANAG 2511).
* **Key Features:**
    * Evidence Scoring (A1-F6).
    * "Red Team" Logic Auditing.
    * Strict separation of Source vs. Information.
* **Best For:** Legal briefs, policy papers, complex investigations.

## Usage
**For AI Agents:**
1.  Scan the `Templates/` directory in this workspace.
2.  Ask the user which template they require.
3.  **COPY** the selected template folder to a new directory (do not modify the originals).
4.  Initialize by reading the `AI_INSTRUCTIONS.md` inside the new project folder.
