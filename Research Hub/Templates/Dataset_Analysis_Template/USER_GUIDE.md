## How It Works

### Phase 1: Setup (AI Creates Structure)

AI will:
1. Ask you about your project
2. Create directory structure
3. Generate documentation templates
4. Set up configuration files
5. Organize your data properly

**You provide:**
- Project name
- Research question
- Data files
- Analysis goals

### Phase 2: Documentation (AI Helps You Document)

AI will:
1. Guide you through each document
2. Help you describe your data
3. Document your decisions
4. Track your process
5. Maintain quality standards

**You provide:**
- Information about your data
- Decisions you make
- Interpretations of findings

### Phase 3: Analysis (AI Writes Scripts)

AI will:
1. Write analysis scripts
2. Clean your data
3. Run statistical analyses
4. Generate visualizations
5. Document everything

**You provide:**
- What analyses you want
- How to interpret results
- Which findings matter most

### Phase 4: Finalization (AI Completes Project)

AI will:
1. Complete all documentation
2. Verify reproducibility
3. Check quality standards
4. Prepare for sharing
5. Create final deliverables

**You verify:**
- Everything looks correct
- Results make sense
- Documentation is complete

**Documentation (9 files):**

- Config.py (all settings)
- Clean Dataset.py (data cleaning)
- Analysis scripts (one per analysis)
- All reproducible and documented

**Outputs:**
- Tables (CSV/Excel format)
- Figures (charts and graphs)
- Database (optional SQLite)

## Best Practices

### Working with AI

**Do:**
- Ask questions when unclear

- Review generated files
- Provide clear instructions
- Save progress frequently

**Don't:**
- Assume AI knows your field

**Do:**
- Document as you go
- Keep raw data safe
- Back up regularly
- Review documentation
- Maintain organization

**Don't:**
- Wait to document
- Modify raw data
